# TextInput

Used to input text into forms.

```jsx live
<TextInput value="Nathan Simpson" label="Name" />
```

| Prop        | Options    | Default |
| ----------- | ---------- | ------- |
| isRequired? | boolean    | false   |
| label       | string     |         |
| onChange    | () => void |         |
| value       | string     |         |
