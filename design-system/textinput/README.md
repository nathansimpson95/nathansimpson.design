# TextInput

Used to input text into forms.

```jsx live
<TextInput value="Nathan Simpson" label="Name" isRequired />
```

| Prop        | Options       | Default |
| ----------- | ------------- | ------- |
| isRequired? | boolean       | false   |
| label       | string        |         |
| onChange    | event => void |         |
| value       | string        |         |
