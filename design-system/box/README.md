# Box

Box is a layout component allows convenient usage of design tokens for.

- margin
- padding
- radii
- background colors

```jsx
<Box bg="emphasis" padding="medium">
  <Text as="span">Hello</Text>
</Box>
```

Box is compatible with semantic HTMl tags...

```jsx
<Box as="section">
  <Text as="span">I'm in a Section</Text>
</Box>
```
